# Release 1.0.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Prepare DocScout-MCP for the 1.0.0 release by fixing version injection, correcting stale documentation, and wiring up automated release CI/CD with semantic versioning, release-drafter, and multi-platform binary + Docker publishing.

**Architecture:** Three independent tracks: (1) fix Go build version injection so no version string ever lives in source, (2) correct stale tool counts and placeholder strings in docs, (3) add three GitHub Actions workflows (release-drafter, labeler-on-PRs, release publisher) plus their config files. All tracks are isolated — no shared state, safe to implement in any order.

**Tech Stack:** Go 1.26, `ldflags -X`, GitHub Actions, `release-drafter/release-drafter@v6`, `docker/build-push-action@v6`, `softprops/action-gh-release@v2`, GHCR, Docker Hub

## Task 9: Create Release Publish Workflow

**Files:**
- Create: `.github/workflows/release.yml`

This is the most complex workflow. It triggers on `v*` tag push **and** `release: published` event. It runs three jobs: `build-binaries` (matrix over 5 platform/arch combos), `release-assets` (attaches binaries to the GitHub Release), and `docker` (multi-arch Docker push to GHCR and Docker Hub).

- [ ] **Step 1: Create the workflow file**

Create `.github/workflows/release.yml`:

```yaml
# Copyright 2026 Leonan Carvalho
# SPDX-License-Identifier: AGPL-3.0-only

name: Release

on:
  push:
    tags:
      - 'v*'
  release:
    types: [published]

permissions:
  contents: write
  packages: write

jobs:
  build-binaries:
    name: Build ${{ matrix.os }}/${{ matrix.arch }}
    runs-on: ubuntu-latest
    strategy:
      matrix:
        include:
          - os: linux
            arch: amd64
            ext: ''
          - os: linux
            arch: arm64
            ext: ''
          - os: darwin
            arch: amd64
            ext: ''
          - os: darwin
            arch: arm64
            ext: ''
          - os: windows
            arch: amd64
            ext: .exe
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up Go
        uses: actions/setup-go@v5
        with:
          go-version: '1.26'
          check-latest: true

      - name: Build
        env:
          CGO_ENABLED: '0'
          GOOS: ${{ matrix.os }}
          GOARCH: ${{ matrix.arch }}
        run: |
          go build \
            -ldflags="-s -w -X main.serverVersion=${{ github.ref_name }}" \
            -o "docscout-mcp-${{ matrix.os }}-${{ matrix.arch }}${{ matrix.ext }}" \
            .

      - name: Upload artifact
        uses: actions/upload-artifact@v4
        with:
          name: binary-${{ matrix.os }}-${{ matrix.arch }}
          path: docscout-mcp-${{ matrix.os }}-${{ matrix.arch }}${{ matrix.ext }}

  release-assets:
    name: Attach Release Assets
    needs: build-binaries
    runs-on: ubuntu-latest
    steps:
      - name: Download all binaries
        uses: actions/download-artifact@v4
        with:
          pattern: binary-*
          merge-multiple: true

      - name: Generate checksums
        run: sha256sum docscout-mcp-* > checksums.txt

      - name: Upload to GitHub Release
        uses: softprops/action-gh-release@v2
        with:
          files: |
            docscout-mcp-*
            checksums.txt

  docker:
    name: Push Docker Images
    needs: build-binaries
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Set up QEMU
        uses: docker/setup-qemu-action@v3

      - name: Set up Docker Buildx
        uses: docker/setup-buildx-action@v3

      - name: Login to GHCR
        uses: docker/login-action@v3
        with:
          registry: ghcr.io
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Login to Docker Hub
        uses: docker/login-action@v3
        with:
          username: ${{ secrets.DOCKERHUB_USERNAME }}
          password: ${{ secrets.DOCKERHUB_TOKEN }}

      - name: Build and push multi-arch image
        uses: docker/build-push-action@v6
        with:
          context: .
          platforms: linux/amd64,linux/arm64
          push: true
          build-args: |
            VERSION=${{ github.ref_name }}
          tags: |
            ghcr.io/leonancarvalho/docscout-mcp:${{ github.ref_name }}
            ghcr.io/leonancarvalho/docscout-mcp:latest
            leonancarvalho/docscout-mcp:${{ github.ref_name }}
            leonancarvalho/docscout-mcp:latest
```
