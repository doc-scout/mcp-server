# Client Configuration Examples

This folder contains configuration examples to integrate DocScout-MCP with popular AI clients.

| Client | File |
|---|---|
| VS Code (Copilot Chat) | [vscode.md](vscode.md) |
| GitHub Copilot | [copilot.md](copilot.md) |
| Antigravity (Google) | [antigravity.md](antigravity.md) |
| Gemini CLI | [gemini.md](gemini.md) |
| ChatGPT Desktop | [chatgpt.md](chatgpt.md) |

## Prerequisites

All configurations require:
1. A compiled `docscout-mcp` binary or a running Docker image.
2. Environment variables `GITHUB_TOKEN` and `GITHUB_ORG` properly set.

See the main [README](../README.md) for details on how to generate a secure GitHub Token.
